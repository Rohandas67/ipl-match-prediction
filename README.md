# 🏏 IPL Match Winner Prediction

A machine learning project that predicts IPL match winners using
historical data from 2008–2025.

## 📊 Dataset
- 1090 IPL matches
- Features: Teams, Venue, Toss, Season, Win Rate

## 🤖 Models Used
- Logistic Regression — 53.67%
- Decision Tree — 51.83%
- Random Forest — 50.46%

## 🔍 Key Insights
- Season is the strongest predictor of match outcome
- Toss has minimal impact on winning (~50/50)
- Most matches are won by wickets
